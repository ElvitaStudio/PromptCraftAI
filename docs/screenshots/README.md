## Screenshot guidelines

- Use demo accounts and fictional prompt content.
- Crop unnecessary Telegram and operating-system UI.
- Keep a consistent device size and color theme.
- Prefer PNG images with readable text.
- Hide Telegram IDs, usernames and payment identifiers.
- Never include bot tokens, API keys, `.env` values or private user data.

When screenshots are ready, add them to this directory and embed them in the
main [`README.md`](../../README.md).
